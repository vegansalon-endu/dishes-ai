📋 このプロジェクトについて

このシステムは、異なるAIモデルが選ぶ「日本の代表的な料理」を比較分析するためのツールです。

🎯 目的

  • 各AIの文化的理解度の比較
  • 日本料理の客観的ランキング作成
  • ヴィーガンレシピ開発の基礎データ収集

  • 他国料理への拡張（ジョージア、イタリア、中国、トルコ）
